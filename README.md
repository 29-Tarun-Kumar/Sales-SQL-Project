# 📊 Sales Data Analysis (SQL Project)

## 📌 Project Overview

This SQL project analyzes sales data using MySQL. It focuses on generating sample data, performing aggregations, and extracting business insights using SQL queries like GROUP BY, SUM, and COUNT.
